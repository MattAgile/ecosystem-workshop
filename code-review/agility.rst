Wartości biznesowe
------------------
* Lower "time to market"
* Increase "value to market"
* Increase "quality to market"
* Increase flexibility
* Increase visibility
* Lower costs
* Increase product lifetime

^^^^^^^^^^^^^^^^^^^^
* Test-driven Development
* Evolutionary Design

   * Refactoring
   * CI / CD

* Test-driven Requirements

   * Functional Tests
   * BDD

^^^^^^^^^^^^^^^^^^^^^^^^

   * Functional Tests
   * BDD

   * BDD

   * Refactoring
